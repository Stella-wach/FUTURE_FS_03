#  Quiet Cafe — WordPress Theme

A custom WordPress theme built for **Quiet Cafe**, a cozy, artsy, and family-owned café based in Nairobi, Kenya. This theme was designed and developed as part of an internship project.

---

##  Features

- Custom homepage via `front-page.php`
- Fully responsive design
- Artsy, warm & rustic aesthetic
- Custom menu section
- Reservations form
- Gallery strip
- Location & contact info section
- SEO optimized with schema markup
- Custom admin dashboard
- Block editor support

---

## 🛠️ Built With

- PHP
- WordPress Theme API
- HTML5 & CSS3
- JavaScript
- WordPress Block Editor (Gutenberg)

##  Installation

1. Install [XAMPP](https://www.apachefriends.org) and [WordPress](https://wordpress.org/download)
2. Clone this repository into your WordPress themes folder:
```bash
   git clone https://github.com/YOUR_USERNAME/quiet-cafe-theme.git
```
3. Place the folder in:
```
   wp-content/themes/quiet-cafe-theme
```
4. Go to **WordPress Admin → Appearance → Themes**
5. Activate **Quiet Cafe Theme**
